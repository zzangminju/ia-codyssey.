## Part 1 · 브랜드 전략

### 📌 브랜드 아이덴티티

#### 1. 핵심 키워드: **"우리(We)"**

이 대본 전체의 구조적 장치는 **브랜드명 자체가 철학**이라는 점을 활용합니다.

"우리은행"에서 **'우리'**를 제거하면 차갑고 기능적인 '은행'만 남는다.

'우리(우리말로 We/Us)'는 단순한 고유명사가 아닌, **공동체·관계·연대**를 뜻하는 언어적 장치로 작동합니다. 이름 안에 이미 브랜드 가치가 내재되어 있습니다.

#### 브랜드 포지셔닝

| 대비 구도 | 타 금융 (암묵적) | 우리은행 |
| --- | --- | --- |
| 관계 | 거래(Transaction) | 관계(Relationship) |
| 대상 | 숫자·실적 | 사람 |
| 감성 | 차갑고 기계적 | 따뜻하고 인간적 |
| 역할 | 금융 서비스 제공자 | 삶의 동반자 |

> → **"사람 중심 금융"**이라는 포지셔닝을 경쟁 차별점으로 설정

#### 메인 타겟

20~40대, 금융을 "필요하지만 차갑고 불편한 것"으로 느끼는 사람들

단순한 연령 구분이 아닌 **심리적 타겟**으로 정의하는 게 핵심입니다.

| 구분 | 내용 |
| --- | --- |
| 1차 타겟 | 사회 초년생 / 청년 (22~35세) |
| 2차 타겟 | 소상공인 / 자영업자 (35~50세) |
| 심리적 공통점 | "은행은 나를 숫자로 본다"는 피로감을 가진 사람 |
| 행동적 특징 | 브랜드의 가치관에 공감할 때 선택하는 세대 |

#### 🎨 TONE & MANNER — 어떻게 말하는가

```
차갑지 않게  ────────────────────  과하지 않게
냉소 ×                            눈물 자극 ×
             ↓ 우리은행이 있어야 할 자리
          조용하지만 단단한 온기
```

| 톤 요소 | 방향 |
| --- | --- |
| 언어 톤 | 설득하지 않고 **공감**한다 |
| 감정 온도 | 뜨겁지 않고 **따뜻하게** |
| 시각 톤 | 화려하지 않고 **정갈하게** |
| 음악 톤 | 웅장하지 않고 **서정적으로** |
| 자막 폰트 | 고딕 계열, **신뢰감 + 단정함** |

> ❌ 피해야 할 톤: 눈물 유발형 감동 광고 / 스펙·금리 나열형 / 유머·발랄 계열

#### 💎 USP — 왜 우리은행인가

**Unique Selling Proposition (단 하나의 차별점)**

> "우리은행은 이름 자체가 철학이다. '우리(We)'라는 단어를 브랜드로 가진 유일한 은행."

**이게 이 광고의 USP이자 다른 은행 광고가 절대 쓸 수 없는 무기입니다.**

| 비교 | 타 은행 | 우리은행 |
| --- | --- | --- |
| 브랜드 차별화 방식 | 상품/금리/혜택 강조 | **이름의 의미**로 차별화 |
| 감성 접근 | 행복한 가족 이미지 | **"없다면 어떻게 될까"** 결핍 역설 |
| 공동체 표현 | 카피로만 언급 | 브랜드 네임 자체가 공동체 |

**최종 브랜드 아이덴티티 한 문장**

> "우리은행은 '우리(We)'라는 이름처럼, 숫자가 아닌 사람을 중심에 두고 공동체와 함께 존재하는 금융 파트너다."

#### 광고 목적 / 핵심 메시지

광고 목적/핵심 메세지 : "'우리'라는 이름의 의미를 재발견하게 함으로써, 우리은행을 단순한 금융기관이 아닌 공동체의 동반자로 인식시키는 브랜드 감성 광고."

| 요소 | 내용 |
| --- | --- |
| 광고 목적 | 브랜드 인지 + 감성적 이미지 재정의 |
| 핵심 타겟 인사이트 | "우리은행"을 그냥 은행으로만 알고 있는 사람들 |
| 핵심 메시지 1개 | 우리은행 = 공동체의 동반자 |
| 설득 방식 | 논리가 아닌 **결핍의 감정** ("없다면 어떻게 될까") |

### 🎬 씬별 대본 (원본)

#### 씬 1 10초

- **화면**: AI로 생성한 거대하고 웅장한 우리은행 본점 건물 외관. 그런데 정문에 붙은 "우리은행로고 + 우리은행" 간판에서 "우리은행로고 + 우리" 부분의 빛이 꺼진다. 그냥 "은행"으로 표시됨. 불이 꺼지면서 주변이 어두워지거나 색감이 채도가 빠지며 차갑게 변하는 효과
- **자막/나레이션**: "만약 은행에서 '우리'라는 단어가 사라진다면 어떻게 될까요?"
- **오디오/음악**: 건조하고 낮은 기계음

#### 씬 2 6초

- **화면**: 화면이 전환되며 소상공인, 취약계층, 청년들이 서로를 돕고 미소 짓는 따뜻한 일상이 빠르게 지나감 (AI 생성 스틸컷).
- **나레이션**: "상인의 땀방울을 닦아주고, 청년의 내일을 안아주며, 이웃의 어려움을 함께 나누는 '우리'가 없다면,"

#### 씬 2.5 4초

- **화면**: 차가운 느낌의 은행 내부를 보여주기
- **나레이션**: "은행은 그저 차가운 건물이 될 뿐입니다."

#### 씬 3 5초

- **화면**: 지워졌던 간판에 '우리은행 로고 + 우리'라는 글자가 다시 켜진다. 글자가 따뜻한 황금빛, 부드러운 화이트 톤으로 서서히 차오른다. '우리은행로고 + 우리은행' 완성. 이때 처음에 로고가 없던 자리는 아무것도 쓰지말고 비워놔.
- **나레이션**: "숫자가 아닌 사람을 담는 곳. 거래가 아닌 관계를 쌓는 곳. 우리은행은 언제나 우리의 곁에 있겠습니다"
- **오디오/음악**: 음악 고조, 따뜻한 현악기 선율

#### 씬 4 5초

- **배경**: 따뜻한 화이트/크림 톤
- **화면**: 우리은행 로고 + 우리 마음속 첫번째 금융
- **자막**: "우리은행, 우리 마음속 첫번째 금융"

이때, 모든 씬의 자막은 너무 화려한 폰트보다는 **신뢰감을 주는 고딕 계열의 정갈한 흰색 폰트**를 사용, 자막의 위치는 약간 아래쪽. 
 

 색감 대비 강조: 씬3의 '우리은행' 복구 장면은 **따뜻한 주황/금빛(Warm Tone)**으로 확실하게 대비를 준다.

> "만약 은행에서 '우리'라는 단어가 사라진다면 어떻게 될까요?"

소상공인, 청년, 취약계층이 서로 돕고 미소 짓는 따뜻한 일상 스틸컷이 빠르게 전환. 웜 톤 보정.

- _Veo_: "3~4장 스틸컷 시퀀스, 각 컷에 줌·패닝 모션 부여, 빠른 컷 전환 (컷당 1~2초), 웜 톤 색보정 유지, 총 6초 클립"

**출력 요약**: ① 따뜻한 생활 스틸컷 3~4장(나노바나나) → ② 빠른 컷 전환 동영상 클립 6초(Veo) → ③ 씬 1 나레이션과 연결되는 보이스 파일 이어붙임(ElevenLabs)

#### 씬 2.5 — 차가운 은행 내부  (4초)

#### 씬 3 — "우리" 복구 — 황금빛 점등  (5초)

_썸네일: 황금빛 점등 / 웜 컬러 복구_

**화면 / 나레이션**

꺼졌던 "우리" 글자가 따뜻한 황금빛·화이트 톤으로 서서히 점등. "우리은행" 완성. 로고 자리는 공백 유지. 따뜻한 현악기 고조.

### Q1. 직접 촬영/유료 스톡 소스를 사용하지 않았음을 설명할 수 있는가?

이 광고의 모든 시각·청각 에셋은 **AI 생성 도구만으로 제작**되었으며, 직접 촬영이나 유료 스톡 라이선스는 일절 사용하지 않았습니다.

| 에셋 유형 | 기존 방식 | 이번 프로젝트 방식 |
| --- | --- | --- |
| 건물 외관·실내 이미지 | 현장 촬영 또는 Getty/Shutterstock 구매 | 나노바나나 프롬프트 생성 |
| 인물 (상인·청년·이웃) | 모델 섭외·스튜디오 촬영 | 나노바나나 한국어 프롬프트 생성 |
| 영상 클립 (소등·달리 숏 등) | 촬영 감독·카메라 장비 필요 | Veo 이미지-투-비디오 변환 |
| 나레이션 보이스 | 성우 녹음 스튜디오 | ElevenLabs 한국어 TTS |
| BGM | 음원 라이선스 구매 또는 작곡 의뢰 | Suno 프롬프트 생성 |

이것이 가능했던 핵심 이유는 **나노바나나의 한국어 프롬프트 최적화** 덕분입니다. 서울 도심 건물의 유리 외벽 질감, 한국 전통시장의 조명 색감, 한국인 인물의 자연스러운 표정까지 한국어 프롬프트 한 줄로 제어할 수 있어, 외국산 스톡 이미지가 주는 이질감 없이 로컬 맥락에 밀착된 에셋을 확보했습니다. 저작권 측면에서도 AI 생성물은 스톡 사이트 이용약관이나 초상권 계약에 종속되지 않아, 상업 광고 용도로 자유롭게 활용 가능합니다.

### Q2. '기획 → 생성 → 검수 → 편집/통합' 파이프라인을 어떤 순서로 설계했는지 설명할 수 있는가?

파이프라인은 **에셋 간 의존 관계**를 기준으로 설계했습니다. 앞 단계의 출력물이 다음 단계의 입력이 되는 순서를 엄격하게 지켰습니다.

**① 기획 — 브랜드 전략 확정 후 씬 분해**

브랜드 아이덴티티(핵심 키워드 "우리", USP, 톤앤매너)를 먼저 문서화하고, 그 위에서 씬 대본을 작성했습니다. 씬 설계는 감정 곡선(결핍 → 공허 → 회복 → 각인)을 따라 4개 씬(씬1·2·2.5·3·4)으로 구조화했으며, 각 씬마다 달성해야 할 **목표 한 문장**을 먼저 정의했습니다. 이 목표 문장이 이후 검수 단계의 판단 기준이 됩니다.

**② 생성 — 스틸 → 영상 → 오디오 순서**

```
나노바나나 (스틸컷 생성)
    ↓
Veo (스틸컷 → 영상 클립 변환)
    ↓
구글ai스튜디오 (나레이션 보이스 생성)
    ↓
구글ai스튜디오 (BGM 트랙 생성)
```

영상은 반드시 스틸컷이 먼저 확정된 후에 생성했습니다. Veo의 이미지-투-비디오 방식을 채택한 이유가 바로 이 순서 때문입니다. 텍스트-투-비디오 방식은 결과물의 예측 가능성이 낮아, 스틸컷 단계에서 화면을 확정하고 그 위에 모션을 입히는 방식이 톤 일관성 유지에 유리합니다. 나레이션과 BGM은 영상 클립의 러닝타임이 확정된 후 생성하여 길이를 맞췄습니다.

**③ 검수 — 씬별 목표 대조**

각 씬의 출력물을 ① 기획 단계에서 정의한 목표 문장, ② 톤앤매너 가이드라인(차갑지 않게 / 과하지 않게)의 두 기준으로 대조했습니다. 특히 콜드 톤(씬1·2.5)과 웜 톤(씬2·3·4)의 색감 대비가 명확한지, 나레이션 어조가 감정선을 따라 점진적으로 고조되는지 확인했습니다.

**④ 편집/통합 — 씬 순서에 따라 클립 연결**

각 씬의 출력 요약에 명시된 순서(스틸컷 → 영상 클립 → 나레이션 → BGM)대로 에셋을 취합한 뒤, 영상 편집 툴에서 씬1(10초) → 씬2(6초) → 씬2.5(4초) → 씬3(5초) → 씬4(5초) 순으로 이어 붙여 총 30초 광고를 완성했습니다. BGM은 Suno에서 30초 단일 트랙으로 생성했기 때문에 씬 전환마다 별도 접합 없이 자연스럽게 흐릅니다.

### Q3. 크레딧/대기열 같은 제약을 고려해 씬 수·생성 횟수·대체 도구를 어떻게 설계했는가?

**씬 수 최소화 — 4씬(+1 브릿지) 구조**

총 30초를 씬1(10초)·씬2(6초)·씬2.5(4초)·씬3(5초)·씬4(5초)로 분할했습니다. 씬2.5는 독립 씬이 아닌 씬2와 씬3 사이의 브릿지 컷으로 설계해, 별도 콘셉트 기획 없이 씬1의 비주얼 언어(블루-그레이, 채도 저하)를 그대로 재사용했습니다. 이로써 콘셉트 기획과 프롬프트 작성 횟수를 줄였습니다.

**에셋 재활용으로 생성 횟수 절감**

| 재활용 케이스 | 절감 효과 |
| --- | --- |
| 씬1 소등 스틸컷 → 씬3 모핑 트랜지션의 Before 이미지로 재사용 | 나노바나나 생성 1회 절감 |
| 씬1 구글ai스튜디오 기계음 BGM → 씬3 현악기 BGM과 동일 프로젝트 내 연장 | Suno 신규 프로젝트 개설 불필요 |
| 씬1 구글ai스튜디오 나레이션 보이스 프로필 → 씬2·2.5·3·4 전체 동일 적용 | 보이스 클론 세팅 중복 작업 없음 |

**도구별 단일화로 대기열 리스크 분산**

영상 생성을 Veo 단일 플랫폼으로 통일함으로써, 여러 비디오 AI 플랫폼의 대기열을 동시에 관리해야 하는 부담을 없앴습니다. 나노바나나는 이미지 생성에만, Veo는 영상 변환에만, ElevenLabs와 Suno는 오디오에만 역할을 분리해 도구 간 전환 비용을 최소화했습니다.

**후보 컷 전략으로 리제너레이션 최소화**

씬4 엔딩 슬로건 레이아웃처럼 완성도가 중요한 씬은 나노바나나에서 후보 컷 3장을 한 번에 요청해 선별하는 방식을 택했습니다. 한 장씩 생성하고 마음에 안 들면 재생성하는 방식보다 크레딧 소비가 예측 가능하고, 비교 선택으로 품질 기준도 높일 수 있습니다.

### Q4. 톤앤매너 일관성을 위해 에셋 규격(해상도/비율/색감)과 생성 파라미터를 어떻게 표준화했는가?

**이중 색감 시스템 — 콜드 vs 웜**

광고 전체의 감정 구조를 색감 두 가지로 표현했습니다. 프롬프트에 색감 키워드를 고정 삽입해 씬 전환 시 시각적 대비가 명확히 드러나도록 했습니다.

| 구분 | 적용 씬 | 색감 키워드 (프롬프트 고정값) |
| --- | --- | --- |
| **콜드 톤** | 씬1, 씬2.5 | 블루-그레이, 채도 저하, 냉랭한 색감, 형광등 백색 조명 |
| **웜 톤** | 씬2, 씬3, 씬4 | 웜 오렌지, 골드, 크림/화이트, 따뜻한 간접 조명, 황금빛 |

**해상도·비율 통일**

모든 나노바나나 스틸컷 프롬프트 말미에 "고해상도 스틸컷"을 고정 키워드로 삽입했습니다. Veo 출력 영상은 이미지-투-비디오 방식이므로 입력 스틸컷의 비율을 16:9(방송 광고 표준)로 통일하면 클립 비율이 자동으로 맞춰집니다.

**카메라 워크 파라미터 — 씬 감정에 맞게 고정**

| 씬 | 카메라 워크 | 의도 |
| --- | --- | --- |
| 씬1 | 서서히 줌인 | 간판 소등의 공허함에 집중 |
| 씬2 | 줌·패닝, 컷당 1~2초 | 빠른 일상의 생동감 |
| 씬2.5 | 슬로우 달리 숏 (좌→우) | 냉랭한 공간의 정적 극대화 |
| 씬3 | 서서히 풀아웃 | 황금빛 복구의 벅차오름 |
| 씬4 | 페이드인 → 타이핑 모션 | 안정적인 엔딩 마무리 |

**오디오 파라미터 — BPM·악기 구성 고정**

Suno BGM은 씬1(BPM 55, 기계음, 현악기 없음)에서 씬3(BPM 65→80, 현악기 고조)로 점진적으로 변하는 하나의 세계관 안에서 설계했습니다. 두 구간을 별개 프로젝트로 만들지 않고 동일 세계관임을 프롬프트에 명시해, 전환 구간에서 음악이 이질감 없이 이어지도록 했습니다. ElevenLabs 나레이션은 동일한 보이스 프로필을 전 씬에 유지하되, 씬1(차분·의문 어조) → 씬3(고조된 클라이막스 어조)로 감정 지시어만 변경했습니다.

**자막 규격 고정**

모든 씬에 공통 적용: 고딕 계열 흰색 폰트, 화면 하단 배치, 과도한 강조 효과 없음. 콜드 씬의 차가운 배경에서도, 웜 씬의 황금빛 배경에서도 흰 고딕 폰트는 가독성과 신뢰감을 일관되게 유지합니다.

## Part 4 · 멀티모달 AI 제작 심화 Q&A

### Q6. Text-to-Image와 Image-to-Video(또는 Text-to-Video)의 차이와 본 프로젝트에서 어떤 기준으로 선택했는가?

**세 방식의 구조적 차이**

| 방식 | 입력 | 출력 | 핵심 특성 |
| --- | --- | --- | --- |
| Text-to-Image (T2I) | 텍스트 프롬프트 | 정지 이미지 | 높은 반복 수정 가능성, 결과 예측 어려움 |
| Image-to-Video (I2V) | 이미지 + 텍스트 프롬프트 | 영상 클립 | 시각 앵커 고정 후 모션만 추가, 결과 예측성 높음 |
| Text-to-Video (T2V) | 텍스트 프롬프트 | 영상 클립 | 한 번에 생성, 시각·동작 동시 제어 필요 |

**본 프로젝트의 선택: T2I(나노바나나) + I2V(Veo)의 2단계 방식**

T2V 방식은 '한 번에 생성한다'는 장점이 있지만, 색감·구도·인물 외형·텍스트 레이아웃을 프롬프트 하나로 동시에 제어해야 하므로 수정 비용이 높아집니다. 색감은 맞는데 구도가 틀리거나, 카메라 무브는 맞는데 인물 표현이 어색한 경우 전체를 재생성해야 합니다. 반면 T2I → I2V의 2단계 방식은 정지 이미지 단계에서 시각 요소를 먼저 확정하고, 그 위에 모션만 추가하는 구조입니다.

**씬별 선택 기준**

| 씬 | 핵심 과제 | T2V 대신 I2V를 선택한 이유 |
| --- | --- | --- |
| 씬1 | 간판 소등 + 채도 저하 동시 표현 | 소등 전·후 상태를 스틸컷으로 먼저 확정한 뒤 트랜지션만 생성 → 간판 글자 위치·폰트 오류 방지 |
| 씬2 | 한국인 인물 3~4명 자연스러운 생활 스틸 | 나노바나나의 한국어 최적화로 인물 표현 확정 후 줌·패닝만 추가 → 인물 이질감 최소화 |
| 씬2.5 | 씬1과 동일한 시각 언어 유지 | 씬1 스틸컷과 같은 블루-그레이 톤을 스틸 단계에서 고정 후 달리 숏 생성 |
| 씬3 | 소등→점등 모핑 트랜지션 | Before(소등) · After(점등) 스틸 쌍을 먼저 확보해 모핑 기준점 명확화 |

T2V 방식을 선택했다면 씬3의 '우리은행' 간판 글자가 영상 안에서 왜곡되거나 한국어 폰트가 깨지는 문제가 발생할 가능성이 높았습니다. I2V는 텍스트 레이아웃이 이미 스틸에 반영되어 있기 때문에 이 리스크가 원천적으로 차단됩니다.

### Q7. 도구를 선택할 때 품질/제어/속도/비용 중 무엇을 우선했고 그 이유는?

**우선순위: 제어 > 품질 > 속도 > 비용**

감성 광고에서 브랜드 톤앤매너 이탈은 회복 불가능한 결함입니다. 한국어 간판 텍스트가 왜곡되거나, 콜드 톤과 웜 톤이 역전되거나, 나레이션 어조가 씬 감정선과 어긋나면 아무리 고해상도 결과물이어도 폐기해야 합니다. 따라서 결과물의 방향을 사전에 제어할 수 있는 도구를 최우선으로 선택했습니다.

**도구별 우선 기준**

| 도구 | 1순위 기준 | 근거 |
| --- | --- | --- |
| 나노바나나 | **제어** | 한국어 프롬프트로 한국 건물·인물·생활 맥락을 정밀 지정 가능. 영문 프롬프트 기반 글로벌 도구는 한국적 맥락 재현에 한계 |
| Veo | **제어 + 품질** | I2V 방식으로 스틸 앵커 고정 후 프레임 단위 카메라 워크 지정 가능. 동일 플랫폼 내에서 전 씬 영상 통일 |
| ElevenLabs | **제어** | 보이스 프로필 고정 + 감정 곡선 파라미터(차분→고조) 씬별 조절 가능. 단순 TTS와 달리 어조·강도 세밀 제어 |
| Suno | **제어** | 단일 프로젝트 내에서 기계음→현악기 전환을 BPM과 악기 구성으로 점진적 제어. 세계관 통일성 유지 |

**속도와 비용을 하위에 둔 이유**

속도를 최우선으로 했다면 T2V 단일 생성 방식을 선택했겠지만, 수정 반복 횟수가 늘어 결국 총 소요 시간이 증가합니다. 비용을 최우선으로 했다면 무료 오픈소스 도구(Stable Diffusion, 오픈소스 TTS)를 선택했겠지만, 한국어 맥락 제어와 감정 곡선 조절에 추가 세팅 시간이 필요합니다. 도구 선택 단계에서 제어력을 확보하는 것이 전체 파이프라인의 효율을 높이는 가장 확실한 방법이라고 판단했습니다.

### Q8. 캐릭터/스타일 일관성을 위해 어떤 제어 방법(레퍼런스 고정, 파라미터, 후처리 등)을 사용했는가?

일관성 확보를 **생성 전 단계(프롬프트 설계)**, **생성 중 단계(레퍼런스 고정)**, **생성 후 단계(후처리 지침)** 세 층위로 나누어 관리했습니다.

**① 생성 전 — 프롬프트 공통 키워드 고정**

모든 씬의 프롬프트에 공통 삽입한 키워드를 사전에 정의하고, 씬별로 추가 키워드만 붙이는 방식을 취했습니다.

| 레이어 | 공통 고정 키워드 | 씬별 추가 키워드 |
| --- | --- | --- |
| 이미지 품질 | 고해상도 스틸컷, 사실감 | (없음) |
| 콜드 씬 색감 | 블루-그레이, 채도 저하, 냉랭한 색감 | 씬1: 형광등 백색 / 씬2.5: 공허한 공간감 |
| 웜 씬 색감 | 웜 오렌지, 간접 조명, 따뜻한 | 씬2: 자연광 / 씬3: 황금빛·골드 / 씬4: 크림/화이트 |
| 건물 외관 | 우리은행 본점 건물 외관, 서울 도심 | 씬1: 소등 상태 / 씬3: 황금빛 점등 상태 |

**② 생성 중 — 레퍼런스 이미지 재사용으로 시각 앵커 고정**

씬1에서 생성한 건물 외관 스틸컷을 씬3의 I2V 입력 이미지(Before 상태)로 직접 재사용했습니다. 같은 이미지를 기반으로 모핑 트랜지션을 생성했기 때문에 건물 구조, 간판 위치, 카메라 각도가 자동으로 일치합니다. 씬 간 동일 건물을 각각 새로 생성했다면 외벽 질감·창문 위치·조명 방향이 미묘하게 달라졌을 것입니다.

ElevenLabs에서도 동일한 원리를 적용했습니다. 씬1에서 설정한 보이스 프로필(중저음, 신뢰감, 한국어 남성)을 씬2·2.5·3·4 전체에 동일하게 적용하고, 감정 지시어(차분·의문 → 고조·클라이막스)만 씬별로 변경했습니다.

**③ 생성 후 — 후처리 지침 사전 정의**

씬 간 색온도 차이가 남아 있을 경우를 대비해 편집 단계의 보정 방향을 미리 정의했습니다. 콜드 씬은 블루 채널을 강조하고 채도를 낮추는 방향으로, 웜 씬은 오렌지-골드 채널을 강조하는 방향으로 기준을 통일했습니다. 자막은 모든 씬에서 고딕 계열 흰색 폰트 + 화면 하단 배치로 고정해 편집자 재량으로 변경되는 것을 방지했습니다.

### Q9. 서로 다른 도구 결과물을 통합할 때 생긴 불일치(해상도/비율/색감/톤)를 어떤 기준과 절차로 보정했는가?

불일치는 **예방(프롬프트 단계)** 과 **교정(편집 단계)** 두 단계로 대응했습니다. 불일치가 생긴 후 고치는 것보다 생기지 않도록 설계하는 것이 우선이었습니다.

**예방 — 생성 단계에서 불일치 원인 차단**

| 불일치 유형 | 예방 방법 |
| --- | --- |
| 해상도 불일치 | 나노바나나 전 씬 프롬프트 말미에 "고해상도 스틸컷" 고정 삽입 |
| 비율 불일치 | 스틸컷 입력 단계에서 16:9 비율 통일 → Veo I2V 출력도 자동으로 16:9 유지 |
| 색감 불일치 | 콜드/웜 이중 시스템으로 씬별 색감 키워드 사전 정의, 씬 간 혼입 방지 |
| 음량 불일치 | ElevenLabs 나레이션과 Suno BGM의 볼륨 비율(나레이션 우선)을 편집 기준에 사전 명시 |

**교정 — 편집 단계의 보정 절차**

프롬프트 설계만으로 완전히 막을 수 없는 불일치는 편집 단계에서 아래 순서로 교정했습니다.

```
1단계: 각 씬 클립 색상 히스토그램 비교
    ↓
2단계: 콜드 씬(씬1, 씬2.5) — 블루 채널 +10~15%, 채도 -20% 기준 적용
        웜 씬(씬2, 씬3, 씬4)  — 오렌지-골드 채널 +10%, 노출 +5% 기준 적용
    ↓
3단계: 씬 전환 구간에서 색온도 급변 여부 확인 (씬1→씬2, 씬2.5→씬3이 핵심 전환 포인트)
    ↓
4단계: BGM 볼륨 엔벨로프 조정 (씬1 기계음 -3dB, 씬3 현악기 페이드인 구간 확인)
    ↓
5단계: 나레이션-영상 타임코드 동기화 (씬별 출력 요약의 클립 길이 기준으로 정합)
```

**씬 전환 구간 색감 대비의 의도적 비보정**

### Q12. 만약 60초 버전을 15초로 줄여야 한다면 어떤 씬을 유지/삭제하고 메시지를 어떻게 재구성할지?

> 현재 광고는 30초이므로, 30초 원본을 기준으로 15초 버전을 도출하는 방식으로 답변합니다. 60초 버전을 가정하더라도 핵심 씬 구조와 삭제 논리는 동일하게 적용됩니다.

**씬 유지/삭제 판단 기준**

15초로 압축할 때 보존해야 할 것은 광고의 감정 구조(결핍 → 회복 → 각인)입니다. 이 구조만 유지되면 메시지 전달이 가능합니다.

| 씬 | 30초 버전 | 15초 버전 | 판단 이유 |
| --- | --- | --- | --- |
| 씬1 (간판 소등) | 10초 → **3초로 단축** | 유지 | 광고 전체 전제(결핍)를 설정하는 유일한 씬. 생략 불가 |
| 씬2 (따뜻한 일상) | 6초 → **삭제** | 삭제 | 설명적 씬. 결핍→회복의 감정 구조만 있으면 없어도 메시지 전달 가능 |
| 씬2.5 (차가운 은행) | 4초 → **삭제** | 삭제 | 씬1의 감정을 반복·강화하는 씬. 씬1 단독으로 같은 기능 수행 가능 |
| 씬3 (황금빛 복구) | 5초 → **7초로 확장** | 유지 + 확장 | 광고의 감동 클라이막스. 15초 버전에서 핵심 나레이션을 이 씬이 모두 담아야 함 |
| 씬4 (엔딩 슬로건) | 5초 → **5초 유지** | 유지 | 브랜드 각인. 생략하면 광고가 미완성으로 느껴짐 |

**15초 재구성안**

```
씬1 (3초): 간판에서 "우리" 소등
           나레이션: "은행에서 '우리'가 사라진다면,"

씬3 (7초): 황금빛 점등 → "우리은행" 완성
           나레이션: "숫자가 아닌 사람을 담는 곳.
                     거래가 아닌 관계를 쌓는 곳.
                     우리은행은 언제나 우리의 곁에 있겠습니다."

씬4 (5초): 로고 + 슬로건
           자막: "우리은행, 우리 마음속 첫번째 금융"
```

**메시지 재구성 논리**

씬2와 씬2.5를 삭제하면 '공동체의 구체적 사례(상인·청년·이웃)' 설명이 빠집니다. 이를 보완하기 위해 씬3의 나레이션을 "숫자가 아닌 사람을, 거래가 아닌 관계를"이라는 대비 문장으로 압축해 공동체 가치를 함축적으로 전달합니다. 30초 버전이 '보여주기(show)'로 설득했다면, 15초 버전은 '말하기(tell)'의 비중을 높여 같은 메시지를 밀도 있게 전달하는 방식으로 전환합니다.

씬1을 10초에서 3초로 단축할 때는 소등 장면의 슬로우 모션을 제거하고 컷 전환으로 대체합니다. 대신 씬3에 할당 시간을 7초로 늘려 나레이션 전달 여유를 확보합니다. 결핍의 '충격'보다 회복의 '감동'에 무게 중심을 두는 방향으로 감정 곡선을 재설계하는 것입니다.

### 씬3 프롬프트 비교

**A (원본)**

> "지워졌던 간판에 '우리은행 로고 + 우리'라는 글자가 따뜻한 빛을 내며 다시 채워짐. '우리은행로고 + 우리은행' 완성.
